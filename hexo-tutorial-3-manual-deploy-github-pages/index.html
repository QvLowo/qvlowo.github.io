дёҠдёҖзҜҮпјҡе»әз«ӢдҪ зҡ„з¬¬дёҖеҖӢblogиҲҮжӣҙжҸӣжјӮдә®зҡ„дё»йЎҢ

еүҚиЁҖ

дёҠдёҖзҜҮпјҢжҲ‘еҖ‘жҸҗеҲ°дәҶеҰӮдҪ•дҪҝз”Ёhexoжһ¶иЁӯеҖӢдәәblogпјҢдҪҶжӘ”жЎҲйғҪйӮ„жҳҜеңЁжң¬ең°пјҢйҖҷд»ЈиЎЁеҸӘжңүдҪ зҡ„йӣ»и…ҰжүҚзңӢеҫ—еҲ°blogз¶Ій ҒпјҢ
йӮЈиҰҒжҖҺйәји®“еҲҘдәәд№ҹзңӢеҫ—еҲ°дҪ зҡ„blogе‘ўпјҹ
йҖҷжҷӮеҖҷе°ұиҰҒдҫҶиҒҠиҒҠйғЁзҪІдәҶпјҒ
з°Ўе–®дҫҶиӘӘпјҢйғЁзҪІе°ұжҳҜе°ҮдҪ жң¬ең°зҡ„з¶Із«ҷжӘ”жЎҲпјҢзҷјеёғеҲ°дёҖеҖӢе…¬й–Ӣзҡ„з¶Іи·ҜдјәжңҚеҷЁдёҠгҖӮ
жң¬зҜҮж•ҷеӯёжңғеё¶дҪ дҪҝз”Ёе®Ңе…Ёе…ҚиІ»зҡ„GitHub PagesжңҚеӢҷпјҢе®ҢжҲҗдҪ зҡ„з¬¬дёҖж¬Ўз¶Із«ҷйғЁзҪІпјҒ

з¬¬дёҖжӯҘпјҡе»әз«Ӣ GitHub Repository

йҰ–е…ҲпјҢжҲ‘еҖ‘йңҖиҰҒеҲ°GitHubдёҠе»әз«ӢдёҖеҖӢе°Ҳй–Җеӯҳж”ҫз¶Із«ҷжӘ”жЎҲзҡ„RopositoryгҖӮ
йҖҷеҖӢRepositoryзҡ„еҗҚзЁұжңүеӣәе®ҡж јејҸпјҢдёҖе®ҡиҰҒжҢүз…§йңҖжұӮпјҢдёҚиҰҒиҮӘе®ҡзҫ©гҖӮ

RepositoryеҗҚзЁұпјҡ<дҪ зҡ„еёіиҷҹ>.github.io

и«Ӣе…Ҳзҷ»е…ҘдҪ зҡ„GitHubеёіиҷҹпјҢдёҰе»әз«ӢдёҖеҖӢж–°зҡ„RepositoryгҖӮ

е»әз«ӢжҷӮжңү 2 еҖӢйҮҚй»һиҰҒжіЁж„Ҹпјҡ

  1. еёіиҷҹеӨ§е°ҸеҜ«пјҡдёҚи«–дҪ зҡ„ GitHub еёіиҷҹеҺҹе§ӢеӨ§е°ҸеҜ«зӮәдҪ•пјҢRepositoryзҡ„еҗҚзЁұеӢҷеҝ…е…ЁйғЁдҪҝз”Ёе°ҸеҜ«гҖӮдҫӢеҰӮпјҢеёіиҷҹжҳҜ MyNameпјҢйӮЈеҗҚзЁұе°ұиҰҒиЁӯзӮә myname.github.ioгҖӮ
  2. е…¬й–ӢпјҡRepositoryеҝ…й ҲиЁӯе®ҡзӮәPublicпјҢйҖҷжЁЈGitHub Pagesзҡ„жңҚеӢҷжүҚиғҪи®ҖеҸ–дҪ зҡ„жӘ”жЎҲпјҢдёҰе°Үз¶Із«ҷе…¬й–ӢзөҰжүҖжңүдәәзҖҸиҰҪгҖӮ

еҰӮжһңе·Із¶“е»әз«ӢйҒҺеҗҢеҗҚзҡ„RepositoryпјҢзі»зөұжңғеҮәзҸҫзҙ…иүІиӯҰе‘ҠжҳҜжӯЈеёёзҡ„(еғҸең–зүҮзӨәзҜ„зҡ„йҖҷжЁЈпјүгҖӮз¬¬дёҖж¬Ўе»әз«ӢжҷӮпјҢжңғжҳҜз¶ иүІзҡ„жҸҗзӨәеӯ—гҖӮ

е»әз«Ӣе®ҢжҲҗеҫҢпјҢжҲ‘еҖ‘еҸ–еҫ—еҖүеә«йҖЈзөҗпјҢжҺҘи‘—е°ұеҸҜд»Ҙжә–еӮҷе°Үжң¬ең°зҡ„Hexoз¶Із«ҷйғЁзҪІдёҠеҺ»дәҶгҖӮ

з¬¬дәҢжӯҘпјҡжүӢеӢ•йғЁзҪІпјҚдҪҝз”Ё Hexo жҢҮд»Өзҷјеёғз¶Із«ҷ

Hexoе…§е»әдәҶйқһеёёж–№дҫҝзҡ„дёҖйҚөйғЁзҪІжҢҮд»ӨгҖӮеңЁеҹ·иЎҢд№ӢеүҚпјҢжҲ‘еҖ‘еҸӘйңҖиҰҒдҝ®ж”№дёҖеҖӢиЁӯе®ҡжӘ”пјҢе‘ҠиЁҙHexoиҰҒе°Үз¶Із«ҷйғЁзҪІеҲ°е“ӘиЈЎгҖӮ
йҰ–е…ҲпјҢиҰҒе…Ҳжү“й–ӢHexoе°ҲжЎҲж №зӣ®йҢ„дёӢзҡ„_config.ymlжӘ”жЎҲпјҢеҲ°жңҖдёӢж–№жүҫеҲ° Deployment зҡ„еҚҖеЎҠпјҢдёҰдҝ®ж”№жҲҗд»ҘдёӢе…§е®№пјҡ

йҖҷйӮҠиҰҒжҠҠ<дҪ зҡ„еёіиҷҹ>жӣҝжҸӣжҲҗдҪ иҮӘе·ұзҡ„GitHubеёіиҷҹгҖӮ
repoж¬„дҪҚе°ұжҳҜдҪ еүӣеүӣе»әз«Ӣзҡ„Repositoryзҡ„HTTPSз¶ІеқҖгҖӮ

иЁӯе®ҡе„ІеӯҳеҫҢпјҢеңЁterminalијёе…Ҙд»ҘдёӢдёүеҖӢжҢҮд»Өпјҡ

  1. hexo clean (еҸҜз°ЎеҜ«зӮә hexo cl)

    йҖҷеҖӢжҢҮд»Өжңғжё…йҷӨеҝ«еҸ–жӘ”жЎҲпјҲdb.jsonпјүе’Ңд№ӢеүҚе·Із”ўз”ҹзҡ„йқңж…ӢжӘ”жЎҲиіҮж–ҷеӨҫпјҲpublicпјүпјҢзўәдҝқжҜҸж¬ЎйғЁзҪІйғҪжҳҜеҫһжңҖд№ҫж·Ёзҡ„зӢҖж…Ӣй–Ӣе§ӢгҖӮ

  2. hexo generate (еҸҜз°ЎеҜ«зӮә hexo g)

    Hexo жңғж №ж“ҡдҪ зҡ„ж–Үз« е’Ңдё»йЎҢиЁӯе®ҡпјҢз”ўз”ҹж•ҙеҖӢз¶Із«ҷзҡ„йқңж…ӢHTMLгҖҒCSSе’ҢJavaScriptжӘ”жЎҲпјҢдёҰж”ҫеңЁpublicзҡ„иіҮж–ҷеӨҫдёӯгҖӮ

  3. hexo deploy (еҸҜз°ЎеҜ«зӮә hexo d)

    йҖҷеҖӢжҢҮд»Өжңғе°ҮpublicиіҮж–ҷеӨҫе…§зҡ„жүҖжңүжӘ”жЎҲпјҢpushеҲ°дҪ еңЁ_config.ymlдёӯиЁӯе®ҡеҘҪзҡ„GitHub RepositoryиҲҮжҢҮе®ҡзҡ„еҲҶж”ҜгҖӮ

еҹ·иЎҢе®Ңhexo dеҫҢпјҢзӯүеҫ…е№ҫеҲҶйҗҳеҫҢжү“й–ӢзҖҸиҰҪеҷЁпјҢијёе…Ҙз¶ІеқҖhttps://<дҪ зҡ„еёіиҷҹ>.github.ioпјҢе°ұеҸҜд»ҘзңӢеҲ°з¬¬дёҖеҖӢblogиӘ•з”ҹе•Ұ~

жң¬зҜҮзёҪзөҗиҲҮдёӢдёҖжӯҘй җе‘Ҡ

еҲ°зӣ®еүҚзӮәжӯўпјҢжң¬зҜҮд»Ӣзҙ№дәҶеҰӮдҪ•е»әз«ӢHexoз¶Із«ҷпјҢдёҰйҖҸйҒҺжүӢеӢ•жҢҮд»Өе°Үе®ғжҲҗеҠҹйғЁзҪІеҲ°з¶Іи·ҜдёҠгҖӮйҖҷеҖӢжөҒзЁӢйӣ–з„¶з°Ўе–®пјҢеҚ»жҳҜзҗҶи§Јз¶Із«ҷе»әзҪ®иҲҮзҷјеёғзҡ„еҹәжң¬жҰӮеҝөгҖӮ
дёҚйҒҺпјҢдҪ еҸҜиғҪжңғзҷјзҸҫпјҢжҜҸж¬ЎеҜ«е®ҢжҲ–дҝ®ж”№дёҖзҜҮж–Үз« пјҢйғҪеҫ—йҮҚиӨҮијёе…Ҙhexo cleanгҖҒgenerateгҖҒdeployйҖҷдёүеҖӢжҢҮд»ӨпјҢд№…дәҶйӣЈе…ҚиҰәеҫ—жңүй»һз№Ғз‘ЈгҖӮ
жңүжІ’жңүжӣҙиҒ°жҳҺгҖҒжӣҙзңҒдәӢзҡ„ж–№жі•е‘ўпјҹз•¶з„¶жңүпјҒ

дёӢдёҖзҜҮж–Үз« пјҢжҲ‘еҖ‘е°Үжңғд»Ӣзҙ№еҰӮдҪ•еҲ©з”ЁGitHub ActionsдҫҶжү“йҖ дёҖеҖӢиҮӘеӢ•еҢ–йғЁзҪІзҡ„жөҒзЁӢгҖӮ
еҸӘиҰҒдҪ е°ҮеҜ«еҘҪзҡ„ж–Үз« pushеҲ°GitHubпјҢеҫҢзәҢзҡ„е»әзҪ®иҲҮйғЁзҪІе·ҘдҪңе°Үжңғе…ЁиҮӘеӢ•е®ҢжҲҗпјҢдёҚз”ЁеҶҚзҷЎзҷЎзӯүеҫ…hexoж…ўж…ўйғЁзҪІе•Ұ~ Ow< b
