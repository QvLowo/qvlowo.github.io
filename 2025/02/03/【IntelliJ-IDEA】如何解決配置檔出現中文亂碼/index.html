【IntelliJ IDEA】如何解決配置檔出現中文亂碼

發表於2025-02-03|更新於2025-05-29|Debugging Journal
|瀏覽量:

properties配置檔出現中文亂碼解法

首先打開 setting

  • MAC 路徑
    IntelliJ IDEA > Settings

  • windows 路徑
    File > Settings

在搜尋框打上 enconding

如圖所示：

把框起的部分改成和圖片一樣

Default encoding for properties files 改為 UTF-8
Transparent native-to-ascii conversion 務必 打勾
這樣子中文才會正常顯示

專案遇到亂碼的情況

可以將圖片上方的

  • Global Encoding
  • Project Encoding
    改為UTF-8
